# Siirwa Millers

A premium multi-page frontend web application for a heritage milling company specializing in locally-sourced flour and specialty grains.

## Built With
* HTML5
* CSS3 (Custom Styling & Bootstrap 5)
* JavaScript (ES6+)

## Project Structure
The application features a unique 8-page architecture tailored for a commercial milling business:
1. **Home** (`index.html`) - Overview of the mill and featured grain products.
2. **About Us** - Our heritage, sustainable sourcing, and traditional milling process.
3. **Products Directory** - Main catalog of what the mill produces.
4. **Baker's Flour** - Dedicated product page for commercial and wholesale bakers.
5. **Specialty Grains** - Product page for whole grains, rye, and custom blends.
6. **Wholesale Inquiry** - Bulk ordering forms for commercial bakeries.
7. **Sustainability** - Profiles on partnering local farms and eco-friendly packaging.
8. **Contact** - Hours, mill location map, and interactive contact form.

## Getting Started
To view this project locally:
1. Clone the repository.
2. Open `index.html` directly in any browser or use the VS Code Live Server extension.
