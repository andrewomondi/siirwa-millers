# Siirwa Millers - Web Platform

This repository contains the official frontend web application platform for Siirwa Millers built with HTML5, Bootstrap 5, and JavaScript.

## Core Milestone Features
* **5-Page Linked Navigation Structure:** Fully unified and operational multi-page experience.
* **Responsive Layouts:** Designed using mobile-first Bootstrap utility classes and components.
* **Interactive Client-Side Search:** Real-time JavaScript-driven catalog filtering.
* **Form Verification Logic:** Conditional JavaScript verification handlers to validate email, name, and message entries on the contact page.
